---

`iskra-x2md` 为 [iskra-data](https://github.com/Qhbee/iskra-data) 提供内容，将其需要的各种格式的资料转换为机器友好的、语义清晰的 **Markdown** 格式。

**Input:** PDF, DOCX, EPUB, HTML, TXT and more  
**Output:** clean, structured GitHub Flavored Markdown (GFM)
